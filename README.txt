A plugin to send emails through the Postal Mail API.

== Description ==

This plugin allows WordPress to send emails using the Postal Mail API.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/postal-mail` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Postal Mail screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin require the Postal Mail API? =

Yes, it does. You need to provide the API credentials in the settings.
